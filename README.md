## راهنمای اجرا
### پیش‌نیازها
- نصب Java Development Kit (JDK 8+)
- درایور SQLite JDBC برای اتصال به پایگاه داده

### نحوه اجرا
برای اجرای بازی کافیست کلاس `GameMain.java` را کامپایل و اجرا کنید. پوشه و فایل پایگاه داده به صورت خودکار در اولین اجرا ساخته می‌شوند.

## کنترل‌های بازی
- **حرکت:** استفاده از کلیدهای `W`, `A`, `S`, `D` یا کلیدهای جهت‌نما (Arrow Keys)
- **شلیک:** کلید `Space`

## پایگاه داده
این پروژه از **SQLite** برای مدیریت داده‌ها استفاده می‌کند.
- **مسیر فایل:** `data/game_database.db`
- **جداول:**
    - `users`: شامل اطلاعات کاربری (نام کاربری، رمز عبور، رکوردها و تنظیمات)
    - `game_records`: شامل جزئیات رکوردهای ثبت شده در هر بازی
